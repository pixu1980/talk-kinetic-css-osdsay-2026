@property + rAF orbit trailing

The intro doodle combines a CSS animatable custom property for hue cycling with JS driven drag, auto-motion on a lemniscate curve, and trailing orbits via requestAnimationFrame. 
      
@property --doodle-hue {
  syntax: '<number>';
  inherits: true;
  initial-value: 210;
}

@keyframes hue-cycle {
  to { --doodle-hue: 570; }
  /* 360 full cycle + 210 start */
}

.stage {
  animation: hue-cycle 6s linear infinite;
  --doodle-title-x: 50%;
  --doodle-title-y: 50%;
}

.title {
  background: linear-gradient(90deg,
    oklch(0.75 0.2 calc(var(--doodle-hue) * 1deg)),
    oklch(0.75 0.2 calc((var(--doodle-hue) + 120) * 1deg)));
  background-clip: text;
  color: transparent;
  cursor: grab;
  ...
}

.orbit {
  left: var(--doodle-orbit-x, var(--doodle-title-x));
  top: var(--doodle-orbit-y, var(--doodle-title-y));
  border-color: oklch(0.7 0.15
    calc(var(--doodle-hue) * 1deg) / 0.3);
}

.orbit--inner  { --doodle-orbit-rotate:  5deg; }
.orbit--middle { --doodle-orbit-rotate: 10deg; }
.orbit--outer  { --doodle-orbit-rotate: 15deg; }
.orbit--fourth { --doodle-orbit-rotate: 20deg; }
  • @property registers --doodle-hue as a number so CSS can interpolate it in @keyframes
  • The gradient picks up the animated hue to cycle colors smoothly
  • Orbits read --doodle-orbit-x/y which JS updates with trailing interpolation
  • Each orbit ring has a different rotation for visual depth
JS drives the motion: a lemniscate (figure 8) path for auto-motion, pointer drag, and per-orbit lerp trailing at decreasing speeds. 
      
const trailConfigs = [
  { selector: '.orbit--inner',  speed: 0.27 },
  { selector: '.orbit--middle', speed: 0.20 },
  { selector: '.orbit--outer',  speed: 0.13 },
];

// Orbit trailing: each ring lerps toward the title
const animateOrbitTrail = () => {
  orbitTrail.forEach((orbit) => {
    orbit.currentX +=
      (orbit.targetX - orbit.currentX) * orbit.speed;
    orbit.element.style.setProperty(
      '--doodle-orbit-x', `${orbit.currentX}%`);
  });
  requestAnimationFrame(animateOrbitTrail);
};

// Lemniscate auto-motion (figure 8 curve)
const angle = phase * Math.PI * 2;
const denom = Math.sin(angle) ** 2 + 1;
const rawX  = Math.cos(angle) / denom;
const rawY  = (Math.sin(angle) * Math.cos(angle))
              / denom;
setPosition(50 + rawX * 36, 50 + rawY * 28);

// Drag writes CSS custom properties directly
title.addEventListener('pointermove', (e) => {
  const rect = stage.getBoundingClientRect();
  const px = ((e.clientX - rect.left) / rect.width)
             * 100;
  const py = ((e.clientY - rect.top) / rect.height)
             * 100;
  setPosition(px, py);
});
  • Each orbit has a decreasing speed factor: inner follows fast, outer lags behind
  • The lemniscate parametric equation creates a smooth infinite figure 8 path
  • setPosition() writes --doodle-title-x/y to both stage and title, then updates orbit targets
  • On user pointer interaction, auto-motion stops and drag takes over
  • All visual output is CSS: JS only sets custom properties

@property

Registered custom properties with @property unlock CSS animation on values that would otherwise be static strings. The browser can interpolate typed values like <number>, <angle>, and <color> across keyframes.
